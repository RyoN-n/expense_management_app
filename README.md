# アプリ製作におけるメモ
## 実装したい機能
- ~~月ごとの支出とその内訳~~
- ~~各月の支出（クレジットカード払いの引き落としも考慮）~~
- 個々の支出をリスト化して管理
- 貯金額の目標を設定でき、月々の貯金額に対する支出がどうなのか差分で表示
- 各支出の詳細をみれるようにする

##設計時の注意点
- 月ごとの集計は、各月25日（給料日）から始まり終わるので実際の月の支出額と異なることに注意⇒毎度、グラフを見るたびに実際の支出額と異なり、コードを修正するが、あとから自分の設計思想に気づき、結局何も修正しないことになるので。。。　例11月26日の支出は12月分の支出として扱われ、グラフに反映される。
- 10月、11月はクレカの支出記録が9月終盤からしかしてない関係で、額が少なめになっているが気にしないで。
